# JavaScript Documentation

## Variables
- `let` - Block scoped variable
- `const` - Constant value
- `var` - Function scoped variable

## Data Types
- String
- Number
- Boolean
- Array
- Object
- null
- undefined

## Functions
```javascript
// Regular function
function name() {}

// Arrow function
const name = () => {}

// Function expression
const name = function() {}
```
